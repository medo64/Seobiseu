Shortcut keys

   F5                      Start service
   Shift+F5                Stop service
   Ins                     Add service to list
   Del                     Remove service from list


Version history

Naming

   This program is named after Korean phonetic transliteration of word
   service. In original form it is written as ???. Name was selected
   primarily because it sounds good and I wasn't able to find other
   program by same name. And I was out of better ideas. :)
